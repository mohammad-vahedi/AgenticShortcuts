== COMMON MISTAKES ==

NEVER invent actions. Only use names listed above.
  webContent()    → getWebpageContents()
  askText()       → prompt()
  indexOf()       → use matchText() with capture group
  substring()     → use matchText() with capture group
  getItemAtIndex() → getListItem(@list, index)
  randomChoice()  → getRandomItem(@list)
  range()         → repeat N { }
  dictionary()    → use {"key": "value"} literal
  map/filter/reduce → use for loops

NEVER use # for comments. Use //.
NEVER write bare words on their own line. Wrong: Podcasts.  Right: show("Podcasts")
NEVER use + for string concat. Wrong: "Hello " + @name. Right: "Hello {name}"
NEVER use GET with jsonRequest. Wrong: jsonRequest(url, "GET"). Right: getWebpageContents(url)
NEVER treat user input words as action names. Wrong: openPodcast(). Right: openApp("com.apple.podcasts")

Variables need @ both when defined and used:
  @x = "hello"
  show("{x}")     ← correct
  show("{x}")     ← but also: show(@x) — both work

matchText() rules:
  - Exactly 2 or 3 args: matchText(regex, text, ?caseSensitive)
  - To extract between two markers: matchText("START([\\s\\S]*?)END", @text, false)
    then: @result = getMatchGroup(@matches, 1)
  - Avoid {n,m} quantifiers inside regex strings — Cherri uses {} for interpolation
